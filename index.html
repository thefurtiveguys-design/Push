Bienvenue dans Vercel Deploy Terminal.
Tapez une commande ou utilisez les boutons ci-dessus.

Commandes disponibles:
  • vercel login      - Se connecter à Vercel
  • vercel            - Déployer le projet
  • vercel --prod     - Déployer en production
  • git init          - Initialiser git
  • git add .         - Ajouter les fichiers
  • git commit -m "msg" - Commiter
  • git remote add origin [url] - Lier à GitHub
  • git push          - Pousser vers GitHub

Pour utiliser un dépôt privé GitHub:
  1. Configurez votre token dans le panneau de configuration
  2. Utilisez: git push https://[TOKEN]@github.com/user/repo.git
